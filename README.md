# SpeedReader

Read text extremely fast by displaying one word at a time anchored in one position (RSVP — Rapid Serial Visual Presentation).

## Getting Started

No install needed. Open `index.html` directly in any browser.

1. Paste your text into the textarea
2. Set your desired WPM (words per minute)
3. Click **Start**
